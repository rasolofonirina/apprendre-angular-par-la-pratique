# Apprendre Angular par la pratique

# Installation

```
git clone https://github.com/rasolofonirina/apprendre-angular-par-la-pratique.git
cd apprendre-angular-par-la-pratique/project/gdg-antsirabe
npm i
ng serve -o
```
